📋 Data Binding Summary

1. Interpolation {{ '{' }}{{ '{' }} {{ '}' }}{{ '}' }}

One-way binding from component to view. Displays component data in the template.

2. Property Binding [property]

One-way binding from component to view. Sets element properties with component data.

3. Event Binding (event)

One-way binding from view to component. Responds to user events and calls component methods.

4. Two-way Binding [(ngModel)]

Combines property and event binding. Synchronizes data between view and component.
