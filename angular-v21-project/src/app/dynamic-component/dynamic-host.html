Dynamic Component — createComponent() + outputToObservable()

📋 Template binding (static)

<app-signal-child
  [parentMessage]="value"
  (childMessageEvent)="handler($event)"
/>

Child type is fixed at compile time. Angular resolves inputs/outputs via the template.

⚡ Dynamic creation (runtime)

// 1 — create
const ref = vcr.createComponent(SignalChild);

// 2 — set input
ref.setInput('parentMessage', value);

// 3 — subscribe to output
outputToObservable(ref.instance.childMessageEvent)
  .subscribe(msg => ...);
