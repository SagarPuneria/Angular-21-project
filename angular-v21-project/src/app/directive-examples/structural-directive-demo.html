Custom Structural Directive

appMyStructural is a custom structural directive. It accepts a boolean condition and conditionally stamps content into the DOM using TemplateRef + createEmbeddedView(). 

<ng-template [appMyStructural]="showBox">
  Content appears when showBox is true
</ng-template>

This is equivalent to *ngIf — both demonstrate the same pattern.

How It Works

Pattern What Happens When to Use
@if (condition) { ... } Built-in control flow (Angular 17+) Preferred — simpler syntax, no template wrapper
*ngIf="cond" Structural directive (Angular 2–16) Legacy code, older Angular versions
[appMyStructural]="cond" Custom structural directive (this example) Learning, custom conditions, reusable logic
