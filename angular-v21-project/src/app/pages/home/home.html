What is Angular Routing?

Angular Router maps URL paths to components, enabling a Single Page Application (SPA) to display different views without a full page reload.

Key Concepts

  • Routes – An array of Route objects that map a path to a component.
  • RouterOutlet – A placeholder directive (<router-outlet>) where the matched component is rendered.
  • RouterLink – A directive used on anchor tags to navigate without reloading the page.
  • RouterLinkActive – Adds a CSS class to the active link automatically.
  • provideRouter() – Registers the router at application bootstrap (standalone API).
