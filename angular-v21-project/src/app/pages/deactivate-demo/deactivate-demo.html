How the Guard Works

  1. The route is configured with canDeactivate: [unsavedChangesGuard]
  2. Before leaving, the router calls component.hasUnsavedChanges()
  3. If it returns true, the browser shows a confirm dialog
  4. User clicks OK → navigation proceeds; Cancel → user stays
