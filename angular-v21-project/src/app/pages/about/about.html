How Navigation Works

  • routerLink – Binds a route path to an element.
    <a routerLink="/home">Home</a>
  • routerLinkActive – Applies a CSS class when the route is active.
    <a routerLink="/about" routerLinkActive="active">About</a>
  • Router.navigate() – Programmatic navigation from TypeScript.
    this.router.navigate(['/courses']);
  • redirectTo – Redirects one path to another.
    { path: '', redirectTo: '/home', pathMatch: 'full' }
  • wildcard route – Catches unmatched paths (404).
    { path: '**', component: NotFoundComponent }
